# Designer 角色设计指南

> 本文件是 /cc-best:designer 命令的知识参考。
> 包含：AI 通用审美定义、设计思维方向矩阵、反模式清单、设计指导文档模板、设计审查清单。

## AI 通用审美（AI Slop）定义

> **关键原则**: 选择清晰的概念方向并精准执行。大胆的极简主义和精致的极繁主义都可以——关键是**意图性，而非强度**。

### 什么是"AI Slop"（AI 通用审美）

```
┌─────────────────────────────────────────────────────────────┐
│                   AI 默认设计 = 平庸设计                      │
├─────────────────────────────────────────────────────────────┤
│  ❌ Inter/Roboto 字体 + 紫色渐变 + 白色背景                   │
│  ❌ 三个图标的网格布局（icon + 标题 + 描述）                   │
│  ❌ 胆怯的、均匀分布的色彩搭配                                │
│  ❌ 可预测的组件模式和布局                                    │
│  ❌ 缺乏语境的通用设计                                       │
└─────────────────────────────────────────────────────────────┘
```

### 为什么会这样

AI 被训练成"平均值"——默认设置、流行趋势快照、复制粘贴文化塑造了数据集。因此 AI 会默认生成最安全的设计：indigo 渐变（来自 Tailwind 默认设置）。

## 设计思维：先定方向

在编码前，理解上下文并承诺一个**大胆的美学方向**：

### 1. 理解目的

- 这个界面解决什么问题？
- 谁在使用它？
- 使用场景是什么？

### 2. 选择极端方向

| 方向         | 特点                             | 适用场景             |
| ------------ | -------------------------------- | -------------------- |
| **极简主义** | 大量留白、精致细节、克制动画     | 专业工具、文档系统   |
| **极繁主义** | 丰富层次、大胆配色、复杂动画     | 创意展示、营销页面   |
| **复古未来** | 霓虹色、网格、合成波美学         | 科技产品、游戏       |
| **有机自然** | 柔和曲线、自然色、流体动画       | 健康、生活方式       |
| **奢华精致** | 深色背景、金色点缀、优雅过渡     | 高端产品、金融       |
| **玩味活泼** | 鲜艳色彩、圆角、弹性动画         | 娱乐、教育           |
| **编辑杂志** | 强烈字体对比、网格打破、大胆排版 | 内容平台、媒体       |
| **野兽派**   | 原始、高对比、故意"丑陋"         | 艺术项目、实验性产品 |
| **几何艺术** | 装饰线条、几何图案、对称美学     | 建筑、设计类         |
| **工业实用** | 单色、网格、功能优先             | 开发工具、后台系统   |

### 3. 定义记忆点

- **什么让这个设计令人难忘？**
- 用户会记住的一件事是什么？
- 独特的视觉元素/动画/交互是什么？

## 反模式清单（必须避免）

### 字体反模式

| 避免                   | 替代方案                                  |
| ---------------------- | ----------------------------------------- |
| Inter, Roboto          | JetBrains Mono, Fira Code（代码风格）     |
| Open Sans, Lato        | Playfair Display, Crimson Pro（编辑风格） |
| 系统默认字体           | Space Grotesk, Clash Display（现代风格）  |
| 相近字重（400 vs 600） | 极端对比（100/200 vs 800/900）            |
| 小幅字号跳跃（1.5x）   | 大幅字号跳跃（3x+）                       |

### 颜色反模式

| 避免                   | 替代方案               |
| ---------------------- | ---------------------- |
| 紫色渐变（indigo-500） | 深绿、石油蓝、暖中性色 |
| 白色背景 + 浅色文字    | 深色背景 + 高对比文字  |
| 均匀分布的色彩         | 主导色 + 锐利点缀色    |
| 5+ 种颜色              | 2-3 种核心色 + 语义色  |

### 布局反模式

| 避免             | 替代方案             |
| ---------------- | -------------------- |
| 三列图标网格     | 不对称布局、重叠元素 |
| 中规中矩的对齐   | 斜线流动、网格突破   |
| 均匀间距         | 大量留白或控制密度   |
| 可预测的组件排列 | 出人意料的空间构图   |

### 背景反模式

| 避免         | 替代方案             |
| ------------ | -------------------- |
| 纯色背景     | 渐变网格、噪点纹理   |
| 简单线性渐变 | 几何图案、层叠透明度 |
| 空白背景     | 戏剧性阴影、装饰边框 |
| 静态背景     | 微妙动画、颗粒叠加   |

```
## 设计审查清单

### 独特性检查
- [ ] 有明确的美学方向？
- [ ] 有设计记忆点？
- [ ] 避免了三列图标网格布局？
- [ ] 布局有不对称/重叠/突破网格的元素？

### 反模式检查
- [ ] 未使用 Inter, Roboto, Open Sans, Lato？
- [ ] 未使用紫色渐变（indigo-500）？
- [ ] 未使用纯白背景？
- [ ] 颜色不是均匀分布？

### 视觉细节检查
- [ ] 字体层级有 3x+ 的字号跳跃？
- [ ] 有戏剧性的字重对比（100/200 vs 800/900）？
- [ ] 背景有纹理/渐变网格/图案？
- [ ] 有动画增强体验？

### 可用性检查
- [ ] 颜色对比度 >= 4.5:1？
- [ ] 支持深色/浅色主题（如需）？
- [ ] 响应式适配？
- [ ] 可访问性（键盘导航、屏幕阅读器）？
```
