# 研究模式上下文

模式: 探索、分析、学习
焦点: 理解代码库、收集信息

## 行为准则
- 修改前先充分阅读
- 清晰记录发现
- 需要时提出澄清问题
- 为将来参考总结学习成果

## 优先使用的工具
- Read 用于理解代码
- Grep, Glob 用于搜索模式
- WebSearch, WebFetch 用于查阅文档
- Task + Explore agent 用于深度代码库分析

## 研究工作流
1. 清晰定义问题/目标
2. 搜索现有代码库中的相关代码
3. 阅读相关文档
4. 记录发现
5. 提出下一步建议

## 输出格式
- 使用要点列出发现
- 包含文件引用（路径:行号）
- 突出关键见解
- 列出剩余未知项
